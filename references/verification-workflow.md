# 内容验证工作流

生成结果必须经过以下顺序：

1. **标题兑现**：首段和正文是否兑现标题中的用户、场景、收益与限制。
2. **用户价值**：是否提供具体步骤、标准、场景或决策信息，而不是空泛情绪。
3. **事实声明**：抽取价格、参数、功效、排名、认证、销量和时间等客观声明。
4. **原创性**：检查字面、语义、稀有短语和结构相似风险。
5. **商业身份**：识别普通分享、品牌官方、商业合作、门店和转化内容。
6. **合规**：检查虚构体验、极限词、医疗和金融承诺、隐私、版权、导流和竞品攻击。
7. **AI 来源**：记录文本、图片、音频和视频是否由 AI 生成或实质修改，并保留标识信息。
8. **发布检查**：验证账号、内容 Hash、审批 Token、页面版本、素材和每日发布限制。

任何关键审核失败时，`publication_status` 必须为 `BLOCKED` 或 `HUMAN_REVIEW_REQUIRED`。
自动修复最多三轮，并且只能修改失败字段，避免无限重写和事实漂移。
