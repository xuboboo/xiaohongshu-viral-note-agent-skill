# Enterprise governance reference

Evaluate every operation against tenant, scope, role, region, provider allowlist and account
allowlist. Reserve estimated model/search cost before work, settle actual cost after work and release
reservations on failure. High-risk publishing requires unique approvers, configured quorum,
separation of duties, unexpired approval and exact content-hash binding. Record all administrative,
identity, approval, plugin and publishing decisions in the audit ledger without request bodies or
raw secrets.
